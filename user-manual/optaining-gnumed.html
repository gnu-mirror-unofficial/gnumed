2.1. Obtaining Gnumed software

The latest sources are available by CVS at savannah.gnu.org. If CVS asks for a password, use a blank line. The module name is gnumed

Qouting from GnuMed's CVS instruction

"This project's CVS repository can be checked out through anonymous (pserver) CVS with the following instruction set. The module [gnumed] you wish to check out must be specified as the modulename. When prompted for a password for anoncvs, simply press the Enter key.

You would type:

Source tree: cvs -d:pserver:anoncvs@subversions.gnu.org:/cvsroot/gnumed login

Remeber to hit enter when it asks for a password. Then type:

cvs -z3 -d:pserver:anoncvs@subversions.gnu.org:/cvsroot/gnumed co gnumed

You should then see the cvs directory download into a gnumed directory
