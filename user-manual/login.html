4.1. Logging In

4.1.1. Setting Up Test Account

Before you can start login you will need to create test accounts. The developers have provided such a test account. It is located at gnumed/server/sql/gmTestAccounts.sql. Usually this account will be installed together with the backend tables using bootstrap-gm_db_system.py. There are currently three test accounts: 

	  test-doc (password: test-doc)
	  test-nurse (password: test-nurse)
	  test-secretary (password: test-secretary)

You should use test-doc since this is the least restricted one.

Do *NOT* use these accounts on a production system.
