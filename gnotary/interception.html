Examples for Freeware or Shareware software that can assist in health record tampering:


This section contains only few links to web sites where such tools can be downloaded. The text below the links are original quotes from these web sites, detailing some features of each software. I only grabbed a few example of literally thousands of programs that can be downloaded any time for little or no money. For legitimate purposes, these tools are valuable and very helpful indeed - in fact, most of them are not primarily designed for tampering purposes. Most of these tools just help you to automatize tedious boring tasks.

This site is not meant to encourage anybody to engage in health record tampering. It is to make a point that inauditable tampering can already be achieved by just downloading easy to use tools, and not overly much computer knowledge is required to use them. Of course, one single tool only would often not be enough to annihilate all anti tampering provisions.

An example setup would be similar to this one:

1.)Download gnu parted from http://www.gnu.org/software/parted/, follow the instructions on http://www.gnu.org/software/parted/USER (Paragraph 8, Disk Imaging)

2.) Partition your hard disk, have at least one spare "mirror" partition

3.) make a snapshot of your system onto the mirror partition: now you can travel back in time any time by reinstalling this partition image (or simply booting into it with the help of a bootmanager, such as Smart Boot Manager from http://www.gnuchina.org/~suzhe/

4.) install one of the registry/file monitors from below to check what exactly the software you use changes on the harddsk during normal operation to get a roug idea what to look for

5.) install one of the "macro recorders" tha allow you time stamped logging (& replay) of mouse and keyboard events.

6.) Now you are almost ready for time travel at any time! Mind you, this is a very simplicistic approach, and I personally would not trust it.

It is in the very best interets of any doctor entrusting health records to a computer to know about the security loopholes, and to know what weight his records might have as evidence in court as a consequence of these loopholes. When you see how easy it is to follow steps 1-6, you will agree that you need to be proactive in order to preserve the value of your Electronic Health Records. PC Activity Monitor

(http://www.geocities.com/SiliconValley/Hills/8839/pcacme.html)

PC Activity Monitor (PC Acme) is a powerful security program that tracks and reports all activity on your PC.
It also captures hidden passwords and displays the key(s) that were pushed when entering the password.

PC Activity Monitor Pro (PC Acme Pro) is a powerful yet easy-to-use general-purpose security program that tracks and reports all activity on your PC and optionally emails the activity log to a preconfigured e-mail address.
It records not only all keystrokes, but all window titles and times, clipboard activity, and more. It records your computer startup and shutdown times, as well as the names of all running tasks.
It runs invisibly. You won't see it on the task list, task bar, or system-tray area and has a minimal impact on system performance.



Macro Scheduler

(http://www.mjtnet.com/index.mv?msched6.html)

Macro Scheduler (or MSched) is a powerful scheduling and macro scripting tool for Windows 9x, NT and 2000. Scripts can be built to control any program or command which accepts keyboard and mouse input. Alternatively, mouse and keyboard events can be 'captured' using the built in macro recorder . You can even edit recorded macros! Any number of macros can then be scheduled to take place at a specified time on any day or days of the week, and if required can be repeated at regular intervals


WhatChanged for Windows

(http://www.prismmicrosys.com/whatchanged/)

The program takes and stores snapshots which include detailed information on all files, directories, registry and system configuration information


Advanced Registry Tracer

(http://www.elcomsoft.com/art.html)

Advanced Registry Tracer (ART; formely RegFix) is an utility designed for analysing the changes made to Windows Registry - by making the "snapshots" of it and keeping them in the browsable database. You can compare any two snapshots and get the list of keys/data which are new, deleted or just changed. ART can do comparing not only in the entire Regisrty, but also in any key of the Registry. It is also possible to exclude any keys of the Registry from compare results. Moreover, you can create undo/redo files (for example, to rollback the changes). To view the current state of a key, or to modify it, you can use Jump to Regedit function. Contents of any key can be exported to *.reg file.

Very useful for detecting trojan viruses and elimination some problems caused by software and hardware install/uninstall.

Unlike Registry monitoring software (such as RegMon and Win-Expose Registry), and most uninstallers (CleenSweep, Uninstall, etc.), ART compares full copies of the Registry made at different times, while the software mantioned above just monitors all accesses to the Registry in real time. So, our method allows to track all the changes, and doesn't affect the system performance.

WinSniffer

(http://www.winsniffer.com/)

The most efficient and reliable password sniffer! Win Sniffer is a study in simplicity. Win Sniffer allows network administrators to capture passwords of any network user. Win Sniffer monitor incoming and outcoming network traffic and decode FTP, POP3, HTTP, ICQ, SMTP, Telnet, IMAP, NNTP passwords

Win Sniffer Console has almost the same functionality as a Win Sniffer 1.21, but works in text mode. Additional feature of Win Sniffer Console is a capability to capture outgoing mail (SMTP).


Stealth Keyboard Interceptor

(http://www.geocities.com/SiliconValley/Hills/8839/skinas.html)

Stealth Keyboard Interceptor AutoSender (SKIn AS) is a security program that tracks and reports all activity on your PC. All window titles, keystrokes, menus, mouse clicks, and actions taken are recorded to a password-protected log file and optionally emailed to a preconfigured e-mail address.
 

Stealth Email Redirector

(http://www.geocities.com/SiliconValley/Hills/8839/ser.html)

Stealth Email Redirector (SER) is a program that sends the copies of all outgoing emails. SER monitors outgoing traffic of email client software and intercepts all sending emails. Then program sends out intercepted emails to specified email addresses.



RegSnap

RegSnap is a tool that can help you analyze changes made to the Windows Registry. With it, you can compare saved snapshots and learn which keys were modified, deleted, or added. You can also analyze other sensitive system parts, such as a list of files in Windows and Windows System directories, win.ini and system.ini files, and autoexec.bat and config.sys files. RegSnap able to generate HTML reports, .REG files to undo and redo registry changes. Pro version supports command line interface and OLE Automation (i.e. could be scriptable from your programs).
